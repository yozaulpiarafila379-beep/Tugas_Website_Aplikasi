🧬 4 Pilar OOP

1. Enkapsulasi

Stok dan kondisi barang hanya diubah lewat metode tambahStok() dan kurangiStok().

2. Pewarisan (Inheritance)

Barang extends EntitasGudang. BarangMasuk & BarangKeluar extends Transaksi.

3. Polimorfisme

tampilkanInfo() dan prosesTransaksi() di-override pada turunan.

4. Interaksi Objek

Objek Barang berinteraksi dengan StokBarang dan Transaksi.

                    abstract class EntitasGudang {
                    String kode, nama;
                    EntitasGudang(this.kode, this.nama);
                    void tampilkanInfo();
                    void cetakIdentitas() {
                    print("🔹 $kode — $nama");
                    }
                    }

                    class Barang extends EntitasGudang {
                    String kategori, merk, satuan, kondisi;
                    int jumlahStok;
                    Barang(String kode, String nama, this.kategori, this.merk,
                    this.jumlahStok, this.satuan, this.kondisi) : super(kode, nama);
                    @override void tampilkanInfo() {
                    print("📦 $kode | $nama | $kategori | Stok: $jumlahStok $satuan");
                    }
                    void tambahStok(int jumlah) { jumlahStok += jumlah; }
                    void kurangiStok(int jumlah) {
                    if (jumlahStok >= jumlah) jumlahStok -= jumlah;
                    else print("❌ Stok tidak mencukupi!");
                    }
                    }

1. Class User (Login Sistem)

  • Class User digunakan untuk merepresentasikan data pengguna yang memiliki hak akses terhadap sistem informasi pendataan barang di gudang.
  • Class User memiliki atribut berupa username, password, dan levelUser yang digunakan sebagai informasi autentikasi pengguna.
  • Constructor User() digunakan untuk melakukan inisialisasi data ketika objek User dibuat.
  • Method login() digunakan untuk melakukan proses validasi username dan password pengguna sebelum mengakses sistem.
  • Fungsi main() digunakan untuk membuat objek User dan menjalankan proses autentikasi melalui method login().
  • Hasil proses autentikasi menampilkan informasi mengenai keberhasilan atau kegagalan pengguna dalam melakukan login.

Input:

  • Username pengguna
  • Password pengguna
  • Level akses pengguna

Proses:

  • Membuat objek dari class User menggunakan constructor.
  • Menyimpan data pengguna ke dalam atribut class.
  • Menjalankan method login() untuk melakukan validasi.
  • Melakukan pengecekan kesesuaian data pengguna.

Output:

  • Informasi login berhasil.
  • Informasi login gagal.
  • Menampilkan data pengguna yang berhasil masuk.

Kode Dart:


// ==========================================
// Nama 1 : Yoza Ulpia Rafila
// NIM    : 251410004
//
// Nama 2 : Rifqi Prayoga
// NIM    : 251410005
//
// Kelas  : SI2A
// Sistem Informasi Pendataan Barang Gudang
// ==========================================

class User {

  String username;
  String password;
  String levelUser;


  // Constructor
  User(
    this.username,
    this.password,
    this.levelUser
  );

  // Method Login
  void login(){

    if(username == "admin" && password == "12345"){

      print("Login berhasil");
      print("User : $username");
      print("Level : $levelUser");

    }else{

      print("Login gagal");

    }

  }

}

void main(){

  // Input data user
  var user1 = User(
    "admin",
    "12345",
    "Admin Gudang"
  );

  // Proses login
  user1.login();

}

2. Class Barang (Data Barang)

  • Class Barang digunakan untuk merepresentasikan data barang yang terdapat pada sistem informasi pendataan barang di gudang.
  • Class Barang memiliki atribut berupa kodeBarang, namaBarang, kategori, merk, jumlahStok, satuan, dan kondisi sebagai informasi detail barang.
  • Constructor Barang() digunakan untuk melakukan inisialisasi data barang ketika objek Barang dibuat.
  • Method tampilkanBarang() digunakan untuk menampilkan informasi data barang yang telah tersimpan.
  • Method tambahStok() digunakan untuk menambahkan jumlah stok barang ketika terdapat barang masuk.
  • Method kurangiStok() digunakan untuk mengurangi jumlah stok barang ketika terdapat barang keluar.
  • Hasil proses pengelolaan barang menampilkan informasi data barang dan perubahan jumlah stok.

Input:

  • Kode Barang
  • Nama Barang
  • Kategori
  • Merk
  • Jumlah Stok
  • Satuan
  • Kondisi

Proses:

  • Membuat objek dari class Barang menggunakan constructor.
  • Menyimpan data barang ke dalam atribut class.
  • Menampilkan data barang menggunakan method tampilkanBarang().
  • Menambahkan stok barang menggunakan method tambahStok().
  • Mengurangi stok barang menggunakan method kurangiStok().

Output:

  • Data barang tampil.
  • Jumlah stok bertambah.
  • Jumlah stok berkurang.
  • Informasi kondisi barang.

Kode Dart:


// ==========================================
// Nama 1 : Yoza Ulpia Rafila
// NIM    : 251410004
//
// Nama 2 : Rifqi Prayoga
// NIM    : 251410005
//
// Kelas  : SI2A
// Sistem Informasi Pendataan Barang Gudang
// ==========================================

class Barang {

  String kodeBarang;
  String namaBarang;
  String kategori;
  String merk;
  int jumlahStok;
  String satuan;
  String kondisi;

  // Constructor
  Barang(
    this.kodeBarang,
    this.namaBarang,
    this.kategori,
    this.merk,
    this.jumlahStok,
    this.satuan,
    this.kondisi
  );

  // Method tampilkan barang
  void tampilkanBarang(){

    print("Kode Barang : $kodeBarang");
    print("Nama Barang : $namaBarang");
    print("Kategori : $kategori");
    print("Merk : $merk");
    print("Jumlah Stok : $jumlahStok");
    print("Satuan : $satuan");
    print("Kondisi : $kondisi");

  }

  // Method tambah stok
  void tambahStok(int jumlah){

    jumlahStok += jumlah;

    print("Stok bertambah menjadi $jumlahStok");

  }


  // Method kurangi stok
  void kurangiStok(int jumlah){

    jumlahStok -= jumlah;

    print("Stok berkurang menjadi $jumlahStok");

  }

}

void main(){

  // Input data barang
  var barang1 = Barang(
    "BRG001",
    "Laptop",
    "Elektronik",
    "Asus",
    10,
    "Unit",
    "Baik"
  );


  // Output data barang
  barang1.tampilkanBarang();


  // Proses stok
  barang1.tambahStok(5);
  barang1.kurangiStok(3);

}

Proses:

7. Class Stok Barang

  • Class StokBarang digunakan untuk mengelola dan menghitung total stok barang dalam sistem informasi gudang.
  • Class ini menggabungkan data dari barang masuk dan barang keluar untuk menghasilkan stok akhir.
  • Constructor StokBarang() digunakan untuk menginisialisasi nilai stok awal, barang masuk, dan barang keluar.
  • Method hitungStok() digunakan untuk melakukan perhitungan stok akhir berdasarkan transaksi yang terjadi.
  • Hasil perhitungan digunakan untuk mengetahui kondisi stok barang saat ini.

Input:

  • Stok awal barang
  • Jumlah barang masuk
  • Jumlah barang keluar

Proses:

  • Membuat objek class StokBarang
  • Menyimpan data stok awal
  • Menambahkan barang masuk
  • Mengurangi barang keluar
  • Menghitung stok akhir

Output:

  • Total stok akhir barang
  • Status ketersediaan stok

Kode Dart:


// ==========================================
// Nama 1 : Yoza Ulpia Rafila
// NIM    : 251410004
//
// Nama 2 : Rifqi Prayoga
// NIM    : 251410005
//
// Kelas  : SI2A
// Sistem Informasi Pendataan Barang Gudang
// ==========================================


class StokBarang {

  int stokAwal;
  int barangMasuk;
  int barangKeluar;


  // Constructor
  StokBarang(
    this.stokAwal,
    this.barangMasuk,
    this.barangKeluar
  );


  // Method hitung stok
  void hitungStok(){

    int stokAkhir = stokAwal + barangMasuk - barangKeluar;

    print("Stok Awal : $stokAwal");
    print("Barang Masuk : $barangMasuk");
    print("Barang Keluar : $barangKeluar");
    print("Stok Akhir : $stokAkhir");

    if(stokAkhir <= 0){
      print("Status : Stok Habis");
    } else if(stokAkhir < 10){
      print("Status : Stok Menipis");
    } else {
      print("Status : Stok Aman");
    }
  }

}


void main(){

  // Input data stok
  var stok1 = StokBarang(50, 20, 10);

  // Proses hitung stok
  stok1.hitungStok();

}

8. Class Laporan

  • Class ini berfungsi sebagai media rekapitulasi data seperti barang masuk, barang keluar, dan stok akhir barang.
  • Constructor Laporan() digunakan untuk menginisialisasi data laporan yang akan ditampilkan.
  • Method cetakLaporan() digunakan untuk menampilkan hasil laporan berdasarkan data yang telah diproses sebelumnya.
  • Laporan bertujuan untuk memberikan informasi keseluruhan aktivitas gudang secara terstruktur.

Input:

  • Jenis laporan
  • Periode laporan
  • Data barang masuk
  • Data barang keluar
  • Data stok barang

Proses:

  • Membuat objek dari class Laporan
  • Mengumpulkan seluruh data transaksi gudang
  • Melakukan rekapitulasi data
  • Menjalankan method cetakLaporan()

Output:

  • Laporan data barang
  • Laporan barang masuk
  • Laporan barang keluar
  • Laporan stok akhir

Kode Dart:


// ==========================================
// Nama 1 : Yoza Ulpia Rafila
// NIM    : 251410004
//
// Nama 2 : Rifqi Prayoga
// NIM    : 251410005
//
// Kelas  : SI2A
// Sistem Informasi Pendataan Barang Gudang
// ==========================================

  String jenisLaporan;
  String periodeLaporan;

  Laporan(
    this.jenisLaporan,
    this.periodeLaporan
  );

    print("Jenis   : $jenisLaporan");
